課題名: mini_db
提出ファイル: *.cpp *.hpp
--------------------------------------------------------------------------------

引数としてポート番号と保存先のパスを受け取るプログラムを書いてください。
このプログラムは 127.0.0.1 の指定ポートでクライアント接続を待ち受けます。

プログラムが受信待機の準備を完了したら、標準出力に "ready\n" を表示します。

接続されたクライアントからコマンドを受け取り、キーと値のデータベースを維持します。

接続は永続的です。つまり、1 回のセッションで複数のコマンドを送れます。

使用できるコマンドは次の通りです。

POST:
synopsis:		POST <key> <value>
description:	db に 1 件追加する
return value:	"0"

GET:
synopsis:		GET <key>
description:	db から 1 件取得する
return value:	"0 <value>" または、キーが存在しない場合は "1"

DELETE:
synopsis:		DELETE <key>
description:	db の 1 件を削除する
return value:	キーが正常に削除されたら "0"、存在しなければ "1"

認識できないコマンドが来た場合は "2" を返してください。

キーと値は任意の長さを持ち、空白は含みませんが、1 回の要求は 1000 文字を超えません。

このプログラムは効率的である必要があります。

データ永続化:

SIGINT シグナルを受け取ったとき、現在のデータベース（すべてのキーと値）を引数で指定されたファイルへ保存して終了してください。

プログラム起動時には毎回このファイルの存在を確認し、あれば読み込みます。

ヒント: nc を使ってテストできます。

このディレクトリには、仕組みを思い出すためのネットワークコードのファイルがあります。

例:

$> ./mini_db 1111 .save &
$> nc localhost 1111 | cat -e
POST A B
O$
POST B C
0$
GET A
0 B$
GET C
1$
DELETE A
O$
DELETE C
1$
UNKNOWN_COMMAND
2$
^C
$> pkill -SIGINT mini_db
$> ./mini_db 1111 .save &
$> echo "GET B" | nc localhost 1111 | cat -e
0 C$
S>
