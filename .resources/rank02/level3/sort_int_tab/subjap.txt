課題名           : sort_int_tab
提出ファイル      : sort_int_tab.c
使用可能関数      :
--------------------------------------------------------------------------------

次の関数を書いてください：

void sort_int_tab(int *tab, unsigned int size);

正確に 'size' 個の要素を含む 'tab' int 配列を昇順で（インプレースで）ソートする必要があります。

重複は保持される必要があります。

入力は常に一貫しています。
