課題名           : ft_atoi_base
提出ファイル      : ft_atoi_base.c
使用可能関数      : None
--------------------------------------------------------------------------------

文字列を（基数 N ≤ 16 で表される）受け取り、10進数の整数に変換して返す関数を作成してください。

入力で認識される文字は `0123456789abcdef` です。
もちろん、リクエストされた基数に応じてこれを制限します。
例えば、基数 4 は `0123` を認識し、基数 16 は `0123456789abcdef` を認識します。

大文字も認識しなければなりません: `12fdb3` は `12FDB3` と同じです。

マイナス記号（`-`）は、文字列の最初の文字である場合にのみ解釈されます。

関数は次のように宣言してください:

int	ft_atoi_base(const char *str, int str_base);
